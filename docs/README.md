# LSV Egge Child Theme

Dieses Theme ist die individuelle Erweiterung des Kadence Parent Themes.

## Struktur

- `inc/` PHP-Module
- `css/` Styles
- `js/` JavaScript
- `templates/` spätere Template-Dateien
- `images/` Theme-Grafiken

## Migrationsprinzip

Code Snippets werden einzeln in die passenden Dateien unter `inc/` verschoben.

Ablauf:

1. Code übernehmen
2. Snippet deaktivieren
3. Testen
4. Commit
