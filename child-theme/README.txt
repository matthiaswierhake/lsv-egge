LSV EGGE CHILD THEME 2.0

Installation:
1. ZIP unter Design > Themes > Theme hochladen installieren.
2. Child Theme aktivieren.
3. CSS aus 'Zusätzliches CSS' schrittweise in die passenden Dateien verschieben.
4. Danach entsprechende Blöcke aus 'Zusätzliches CSS' löschen.

CSS-Dateien:
- variables.css: Farben, Abstände, Schatten, Radien
- layout.css: allgemeine Karten, Buttons, Layout-Helfer
- navigation.css: Header, Menü, Benutzer-Menü
- login.css: Loginseite und Loginformular
- redaktion.css: Redaktionsleiste
- startseite.css: Startseite
- mitglieder.css: Mitgliederbereich
- footer.css: Footer
- responsive.css: Tablet und Mobil

PHP:
Der Ordner inc/ ist vorbereitet. Code-Snippets bleiben zunächst aktiv.
Später können wir Snippets schrittweise in inc/*.php verschieben.
